## ğŸš€ Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional dependencies required

### Installation

1. Clone or download this repository
2. Navigate to the `Rock-Paper-Scissors` directory
3. Open `index.html` in your web browser

### How to Play

1. Open the game in your browser
2. Click on any of the three choices (Rock, Paper, or Scissors)
3. The computer will randomly select its choice
4. The game will display:
   - A message indicating the winner
   - Updated scores for both player and computer
   - Color-coded feedback (green for win, red for loss, dark blue for draw)
